# GuardPDF - Free Online PDF Tools

> 100% browser-based. Your files never leave your device.

🔗 **[Try it now → guardpdf.com](https://guardpdf.com)**

## Features

| Tool | Description | Link |
|------|-------------|------|
| 🗜️ Compress PDF | Reduce file size up to 80% | [Try →](https://guardpdf.com/en/compress-pdf) |
| 📎 Merge PDF | Combine multiple PDFs into one | [Try →](https://guardpdf.com/en/merge-pdf) |
| ✂️ Split PDF | Extract specific pages | [Try →](https://guardpdf.com/en/split-pdf) |
| 🔒 Encrypt PDF | Password protect your PDF | [Try →](https://guardpdf.com/en/encrypt-pdf) |
| 🔓 Unlock PDF | Remove PDF password | [Try →](https://guardpdf.com/en/unlock-pdf) |
| 🖼️ PDF to Images | Convert pages to PNG/JPG | [Try →](https://guardpdf.com/en/pdf-to-images) |
| 📸 Images to PDF | Combine images into PDF | [Try →](https://guardpdf.com/en/image-to-pdf) |
| 🗑️ Delete Pages | Remove unwanted pages | [Try →](https://guardpdf.com/en/delete-pdf-pages) |
| 📄 Extract Pages | Pull out specific pages | [Try →](https://guardpdf.com/en/extract-pdf-pages) |
| 📝 Markdown to PDF | Convert MD to PDF | [Try →](https://guardpdf.com/en/markdown-to-pdf) |

## Why GuardPDF?

- 🔒 **Privacy First** — All processing in your browser, no server upload
- ⚡ **Fast** — WebAssembly powered, no waiting for server
- 🆓 **Free** — No signup, no watermark, no limits
- 🌍 **Multilingual** — English & 中文
